1) kiedy powstał JavaScript (podaj datę).

JavaScript został stworzony w grudniu 1995 roku.

2) dlaczego jest to język skryptowy

Skrypty JS są interpretowane podczas przeglądania stron internetowych i nie wymagają kompilacji przed uruchomieniem.

3) gdzie jest wykonywany JS (klient lub serwer)

JavaScript jest wykonywany po stronie klienta, czyli na komputerze użytkownika przez przeglądarkę internetową.

4) dlaczego jest JS to bezpieczne narzędzie

JavaScript ma znacznie ograniczone uprawnienia dostępu do zasobów komputera.

5) czy jest to język obiektowy , jeśli tak to dlaczego.

JavaScript wykorzystuje definicje obiektów, metod, zdarzeń, które mogą być zastosowane do opisu właściwości obiektów.

6) wymień zalety JS:

- Łatwość nauki i zrozumienia.
- Możliwość tworzenia interaktywnych stron internetowych.
- Duża popularność i wsparcie ze strony społeczności programistycznej.

7) Framework to:

Framework to struktura lub platforma programistyczna, która dostarcza gotowe narzędzia i biblioteki do tworzenia aplikacji. Przykłady to Angular, React, Vue.js.

8) Do aplikacji mobilnych można użyć Framework

Tak, do aplikacji mobilnych można użyć frameworków. Przykłady to React Native, Flutter, Ionic.
